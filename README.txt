# Customer Registration
1.0

This application made for hungarian financial advisor. 
You can registrate clients, loans and savings. 
The data is stored in MySQL database.

Functions:

- [x] Graphical interface is Swing,
- [x] Create customer data, 
- [x] Create loan data,  
- [x] Create savings data (LTP in Hungary
- [x] Database MySql (local)
- [x] Import data from Excel (specialized), customer,load,savings data
- [x] List customer with search
- [x] List loans with search
- [x] List savings with search
- [x] List altogether with search
